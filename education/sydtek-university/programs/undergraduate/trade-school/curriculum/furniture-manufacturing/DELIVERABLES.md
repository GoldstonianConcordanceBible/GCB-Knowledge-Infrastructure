# Deliverables
## Furniture Manufacturing Curriculum

Students in the Furniture Manufacturing program are expected to complete and submit deliverables that show both technical growth and production readiness.

---

## Required Deliverables by Program Completion

- workshop safety acknowledgment
- tool competency checklist
- lumber identification exercise
- measurement and layout exercise
- joinery practice board
- table build documentation
- chair build documentation
- rocking chair or motion-based furniture study
- finishing sample board
- defect correction exercise
- capstone build
- GitHub portfolio folder
- final product presentation
- product listing draft

---

## Documentation Standards

Each major deliverable should include:

- title of build or exercise
- date completed
- materials used
- dimensions
- process summary
- problems encountered
- corrections made
- final reflection
- photos when appropriate

---

## Portfolio Requirement

Before graduation, each student must maintain a program portfolio showing progression from foundational exercises to finished commercial work.

The portfolio must demonstrate:

- technical development
- workshop consistency
- ability to complete work
- readiness for employment or entrepreneurship
