# FMT 202
## Table Construction Studio

Course Code: FMT 202  
Course Level: 200  
Term Placement: Term 2 — Structural Furniture Construction

---

## Course Description

FMT 202 introduces students to the structural design and fabrication of tables.

Tables serve as the foundational furniture system because they require:

- flat surfaces
- balanced support
- durable joinery
- aesthetic alignment

Students build a complete table structure during this course.

---

## Course Purpose

The purpose of this course is to transition students from joinery practice into real furniture construction.

Students learn how to plan, assemble, and refine an entire structural furniture piece.

---

## Learning Outcomes

By the end of this course, students will be able to:

- design a simple table structure
- prepare table legs and supports
- construct stable table frames
- attach tabletops correctly
- ensure structural stability
- correct uneven surfaces

---

## Lab Requirements

Students must complete:

- table leg fabrication
- frame assembly
- tabletop preparation
- full table construction
- leveling adjustments

---

## Assessment Criteria

Students are assessed on:

- structural stability
- frame squareness
- tabletop alignment
- finishing preparation
- craftsmanship consistency

---

## Required Deliverables

- table construction plan
- materials list
- cut list
- build documentation
- finished table structure
