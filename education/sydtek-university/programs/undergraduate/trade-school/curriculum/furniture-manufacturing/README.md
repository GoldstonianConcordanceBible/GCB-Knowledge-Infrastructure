# Curriculum
## Furniture Manufacturing
## SydTek University Trade School

This curriculum defines the instructional sequence for the Furniture Manufacturing program.

The curriculum is organized to move students from foundational workshop readiness into structural furniture fabrication, production systems, and commercial capstone work.

---

## Curriculum Design Principles

The curriculum is built on five principles:

1. safety before speed
2. repetition before complexity
3. structure before aesthetics
4. documentation alongside fabrication
5. commercial readiness at program completion

---

## Curriculum Sequence

The program is divided into four instructional terms:

- Term 1 — Foundations of Woodcraft
- Term 2 — Structural Furniture Construction
- Term 3 — Production and Finishing Systems
- Term 4 — Commercial Capstone

---

## Delivery Model

Instruction is delivered through:

- guided demonstrations
- workshop labs
- supervised practice
- project-based builds
- GitHub documentation
- portfolio assessment

---

## Curriculum Outputs

Each student is expected to produce the following across the program:

- safety checklists
- tool competency records
- practice builds
- joinery samples
- materials lists
- cut lists
- build logs
- process photography
- capstone documentation
