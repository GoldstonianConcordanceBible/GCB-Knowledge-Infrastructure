# Data Directory Overview

## Purpose
This directory stores program-level data used for assessment, reporting, and accreditation evidence.

---

## Files and Schema

### enrollment_tracking.csv
Fields:
- StudentID
- Course
- Status

### course_completion.csv
Fields:
- StudentID
- Course
- CompletionStatus

### learning_outcomes_results.csv
Fields:
- Outcome
- Score
- Term

### ai_usage_tracking.csv
Fields:
- StudentID
- Course
- AI_Tool
- Usage_Type

### course_artifact_tracking.csv
Fields:
- Course
- Artifact
- Outcome

---

## Data Status
STATUS: Pre-Launch

These files are templates. Data population will begin upon program launch.

---

## Update Frequency
- Course-level data: each term
- Program-level summaries: annually

---

## Data Use
- Assessment reporting
- Accreditation evidence
- Program improvement
- AI usage monitoring
