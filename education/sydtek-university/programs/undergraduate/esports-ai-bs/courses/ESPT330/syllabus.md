# ESPT 330 – Live Event Production and Streaming Technologies

## Credits
3

## Description
Students produce live esports broadcasts and learn streaming workflows.

## Learning Outcomes
1. Design live streaming setups.
2. Execute production workflows.
3. Apply AI tools in production.
4. Evaluate stream performance.

## AI Integration
AI for automation, clipping, and stream optimization.

## Assignments
- Stream Setup Plan
- Live Production Exercise
- Performance Report

## Assessment
- Setup → CLO1
- Production → CLO2
- Report → CLO4
