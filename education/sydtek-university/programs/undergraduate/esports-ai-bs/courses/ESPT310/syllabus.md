# ESPT 310 – Live Event Management and Intelligent Operations

## Credits
4

## Description
This course focuses on planning, executing, and evaluating esports and live events. Students learn logistics, staffing, budgeting, risk management, and post-event analysis. AI tools are introduced for forecasting attendance, optimizing schedules, and improving operational efficiency.

## Learning Outcomes
1. Plan and manage esports events.
2. Apply operational strategies to live environments.
3. Use AI tools for event planning and forecasting.
4. Evaluate event success using data.

## AI Integration
Students use AI for scheduling simulations, cost estimation, and event scenario modeling.

## Assignments
- Event Plan
- AI Scheduling Simulation
- Post-Event Analysis

## Assessment
- Event Plan → CLO1
- AI Simulation → CLO3
- Analysis → CLO4
