# ESPT 310 – Live Event Management and AI-Driven Operations

## Credits
4

## Delivery Format
Hybrid / MOOC-supported

## Prerequisites
ESPT120

## Course Description
This course develops advanced competencies in planning, executing, and evaluating esports and live events. Students explore logistics, staffing, budgeting, risk management, and operational analytics. Emphasis is placed on integrating AI tools for forecasting, scheduling, resource allocation, and post-event optimization.

## Course Learning Outcomes (CLOs)
1. Design a complete esports event plan.
2. Apply operational frameworks to live event environments.
3. Use AI tools to model event logistics and resource allocation.
4. Evaluate event performance using data and analytics.

## Program Learning Outcome Alignment
- CLO1 → PLO1, PLO6
- CLO2 → PLO1, PLO7
- CLO3 → PLO2, PLO8
- CLO4 → PLO8

## AI Integration Statement
Students will use AI tools to simulate event logistics, generate scheduling models, forecast attendance, and analyze post-event performance. All AI use must be documented and critically evaluated.
