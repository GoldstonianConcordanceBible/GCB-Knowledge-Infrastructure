# Courses Directory

This directory contains all core course folders for the Bachelor of Science in Esports, Gaming, and AI Performance.

## Standard Course Folder Structure
Each course folder should contain:
- syllabus.md
- assignments.md
- assessments.md
- ai_integration.md
- outcomes_alignment.md
- playlist_mapping.csv

## Course Design Rules
- Every course must align with at least one program learning outcome
- Every course must include AI integration
- Every course must include at least one assessable artifact
- Every course must document instructional time expectations
- Every course must preserve evidence for review
