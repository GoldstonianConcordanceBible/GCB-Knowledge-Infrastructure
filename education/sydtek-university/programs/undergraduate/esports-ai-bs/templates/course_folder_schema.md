# Course Folder Schema

Each course folder shall contain the following files:

- syllabus.md
- assignments.md
- assessments.md
- ai_integration.md
- outcomes_alignment.md
- playlist_mapping.csv

Optional supporting files may include:
- rubric.md
- project_brief.md
- reading_list.md
- lab_guide.md
- reflection_prompts.md

## Required Purpose of Each File

### syllabus.md
Defines description, outcomes, schedule, policies, and grading.

### assignments.md
Defines major course assignments and deliverables.

### assessments.md
Defines assessment structure and evidence design.

### ai_integration.md
Defines how AI is used, evaluated, disclosed, and governed in the course.

### outcomes_alignment.md
Maps course learning outcomes to program learning outcomes.

### playlist_mapping.csv
Maps instructional media, playlists, or MOOC content to modules and hours.
