# Playlist Mapping Standard

## Purpose
This standard defines how playlists, media collections, and MOOC-supported instructional resources are mapped into course delivery.

## Required Fields
Each course playlist mapping file should document:
- Course
- Module or week
- Playlist or media title
- URL or internal reference
- Required viewing or engagement hours
- Verification method
- Notes

## Accepted Verification Methods
- Reflection submission
- Quiz
- Discussion post
- Annotation activity
- Embedded knowledge check
- Project integration

## Instructional Time Rule
Playlist or media-supported learning should be documented in a way that allows review of expected instructional engagement for audit and program planning purposes.

## Quality Expectations
Media should be:
- Relevant to stated learning outcomes
- Current or intentionally historical
- Appropriate in length and level
- Connected to measurable student activity
