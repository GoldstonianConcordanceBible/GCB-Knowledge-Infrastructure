# Curriculum Revision Policy

## Purpose
This policy defines how curriculum changes are proposed, reviewed, approved, documented, and implemented.

## Types of Revisions
### Minor Revisions
- Assignment updates
- Weekly topic refinement
- Tool updates
- Example changes
- Reading list updates

### Moderate Revisions
- Course description updates
- Learning outcome revisions
- Assessment redesign
- Technology requirement changes
- Elective list adjustments

### Major Revisions
- Credit-hour changes
- Program outcome revisions
- Required course changes
- Program title changes
- Delivery model redesign

## Approval Structure
- Faculty of record may propose revisions
- Program lead reviews proposed changes
- Curriculum committee approves moderate and major changes
- Institutional approval may be required for major structural changes

## Documentation
All approved revisions should be logged with:
- Date
- File updated
- Reason for change
- Evidence supporting the change
- Approval authority
