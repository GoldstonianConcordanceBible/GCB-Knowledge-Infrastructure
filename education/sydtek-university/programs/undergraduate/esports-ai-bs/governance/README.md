# Governance Overview

## Purpose
This document explains how governance functions operate across the program.

## Governance Components
- curriculum_committee.md → curriculum oversight
- course_approval_workflow.md → course creation and revision
- assessment_policy.md → assessment expectations
- continuous_improvement_cycle.md → review process
- program_review_schedule.md → review timing
- faculty_qualification_framework.md → instructor standards

## Escalation Path
1. Faculty identifies issue
2. Program lead review
3. Curriculum committee review
4. Institutional escalation if required

## Revision Cycle
- Annual review for all governance documents
- Immediate revision for compliance issues

## Relationship to Assessment
Governance decisions are informed by:
- assessment results
- KPI dashboards
- faculty reports
- advisory input

## Status
Active governance framework
