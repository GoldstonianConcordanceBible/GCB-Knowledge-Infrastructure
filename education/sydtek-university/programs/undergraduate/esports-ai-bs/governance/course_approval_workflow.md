# Course Approval Workflow

## Purpose
This workflow defines the minimum review path for adding or revising courses in the program.

## Required Review Steps
1. Draft course documents prepared
2. Course outcomes aligned to program outcomes
3. AI integration documented
4. Assessment structure documented
5. Playlist or media mapping documented when applicable
6. Program lead review
7. Curriculum committee review for new or substantially revised courses
8. Final version stored in approved repository structure

## Minimum Required Files Before Approval
- syllabus.md
- assignments.md
- assessments.md
- ai_integration.md
- outcomes_alignment.md
- playlist_mapping.csv

## Review Questions
- Is the course academically coherent?
- Is the credit-hour expectation reasonable?
- Is AI integration appropriate and ethical?
- Are outcomes measurable?
- Is evidence collection clear?
- Does the course duplicate or strengthen existing offerings?
