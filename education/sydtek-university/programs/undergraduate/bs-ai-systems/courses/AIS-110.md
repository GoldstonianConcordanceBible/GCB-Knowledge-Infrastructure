AIS-110 Course Shell
Programming for AI Systems

Credits: 3  
Level: Undergraduate

Catalog Description

Introduces programming foundations for artificial intelligence systems using languages and tools appropriate for data processing, model development, and reproducible experimentation.

Students learn programming logic, data structures, functions, scripting workflows, and basic AI-oriented software practices.

Prerequisites

None

Course Learning Outcomes

Students will be able to:

1 write basic programs used in AI and data workflows
2 apply variables, functions, control flow, and modular design in technical tasks
3 manipulate data structures commonly used in AI systems
4 create reproducible programming workflows for simple AI-related tasks
5 document code and repository artifacts clearly

GitHub Evidence Statement

Where GitHub-based evidence is used, academic credit is awarded for verifiable documented work aligned to course outcomes.

Repository visibility may be public or private unless otherwise stated, but authorized academic reviewers must be able to inspect required evidence.

Open Discussion Pedagogy Note

This course may include inspectable code review, structured debugging discussion, and evidence-linked programming reflection.

Mirror → Water → Fire Alignment

Mirror: represent code logic and computational structure accurately  
Water: interpret errors, outputs, and workflow behavior  
Fire: build and refine documented programming artifacts for AI use
