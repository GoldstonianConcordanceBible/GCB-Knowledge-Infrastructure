AIS-495 Course Shell
AI Capstone Project

Credits: 6  
Level: Undergraduate

Catalog Description

Students design, implement, evaluate, and present a substantial artificial intelligence system or research project demonstrating undergraduate mastery of AI concepts, engineering workflows, documentation standards, and evidence-based analysis.

Projects must produce inspectable, reproducible, and faculty-reviewable artifacts.

Prerequisites

AIS-490  
Senior standing

Course Learning Outcomes

Students will be able to:

1 design a substantial AI system or research project with clear goals and methods
2 implement documented workflows for data, models, experimentation, and evaluation
3 analyze results, limitations, and system tradeoffs responsibly
4 communicate findings through technical writing, repository documentation, and formal presentation
5 deliver a capstone repository demonstrating undergraduate-level mastery in artificial intelligence systems

GitHub Evidence Statement

Academic credit is awarded for verifiable documented work aligned to course and program outcomes.

Repositories may be public or private unless otherwise stated, but authorized academic reviewers must be able to inspect required evidence.

Capstone repositories should typically include:

- README documentation
- problem statement
- literature or precedent review
- dataset description
- code or notebooks
- model or system artifacts where appropriate
- evaluation results
- limitations discussion
- final report
- presentation materials

Open Discussion Pedagogy Note

This course may include faculty review, inspectable milestone critique, presentation feedback, and revision-based capstone completion workflows.

Mirror → Water → Fire Alignment

Mirror: represent the system, methods, and evidence accurately  
Water: interpret performance, findings, and limitations with intellectual honesty  
Fire: finalize, present, and preserve a substantial AI capstone through inspectable academic artifacts
