AIS-220 Course Shell
Neural Networks

Credits: 3  
Level: Undergraduate

Catalog Description

Introduces neural network architectures including perceptrons, feedforward networks, backpropagation, activation functions, convolutional networks, recurrent networks, and training optimization.

Students analyze how neural systems learn from data and how design choices affect performance.

Prerequisites

AIS-201  
MATH-130

Course Learning Outcomes

Students will be able to:

1 explain the structure and function of neural network architectures
2 describe how backpropagation and optimization support learning
3 train introductory neural models for classification or prediction tasks
4 evaluate neural network performance and model behavior
5 document model development and experimentation clearly

Open Discussion Pedagogy Note

This course may include analysis of model design tradeoffs, benchmark claims, and neural network limitations.

Mirror → Water → Fire Alignment

Mirror: represent model architecture and training processes accurately  
Water: interpret learning behavior, evaluation results, and tradeoffs  
Fire: train and refine neural network systems through documented workflows
