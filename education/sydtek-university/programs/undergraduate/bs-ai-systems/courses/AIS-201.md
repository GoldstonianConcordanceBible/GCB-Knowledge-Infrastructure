AIS-201 Course Shell
Machine Learning Foundations

Credits: 3  
Level: Undergraduate

Catalog Description

Introduces foundational machine learning concepts including supervised learning, unsupervised learning, feature selection, model training, evaluation, and overfitting.

Students examine how data is transformed into predictive systems and how model performance is assessed in reproducible workflows.

Prerequisites

AIS-110  
MATH-140

Course Learning Outcomes

Students will be able to:

1 explain major machine learning paradigms and workflows
2 distinguish supervised, unsupervised, and basic reinforcement learning contexts
3 train and evaluate introductory machine learning models
4 analyze model performance using common evaluation measures
5 document experiments and findings through inspectable academic artifacts

GitHub Evidence Statement

Where GitHub-based evidence is used, academic credit is awarded for verifiable documented work aligned to course outcomes.

Repository visibility may be public or private unless otherwise stated, but authorized academic reviewers must be able to inspect required evidence.

Open Discussion Pedagogy Note

This course may use structured critique of model assumptions, training decisions, benchmark claims, and performance limitations.

Mirror → Water → Fire Alignment

Mirror: represent datasets, models, and evaluation methods accurately  
Water: interpret patterns, performance, and error sources  
Fire: train, refine, and document machine learning workflows
