AIS-120 Course Shell
Data Structures for Intelligent Systems

Credits: 3  
Level: Undergraduate

Catalog Description

Introduces core data structures and algorithmic thinking used in intelligent systems including arrays, lists, stacks, queues, trees, graphs, and search structures.

Students examine how data organization affects reasoning, retrieval, optimization, and performance in AI-related contexts.

Prerequisites

AIS-110

Course Learning Outcomes

Students will be able to:

1 explain core data structures used in computational systems
2 compare the strengths and weaknesses of major data structures
3 apply algorithmic reasoning to basic intelligent system problems
4 analyze how structure affects storage, search, and retrieval behavior
5 document implementations and comparisons through inspectable artifacts
