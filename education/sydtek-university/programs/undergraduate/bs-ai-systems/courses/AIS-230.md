AIS-230 Course Shell
Natural Language Processing

Credits: 3  
Level: Undergraduate

Catalog Description

Introduces natural language processing methods including tokenization, embeddings, text classification, sequence modeling, transformers, and language model evaluation.

Students examine how computational systems represent, interpret, and generate human language.

Prerequisites

AIS-201

Course Learning Outcomes

Students will be able to:

1 explain foundational concepts in natural language processing
2 compare major approaches to representing and modeling text
3 analyze text datasets using basic NLP workflows
4 evaluate NLP outputs and limitations in context
5 document language processing experiments through inspectable artifacts

GitHub Evidence Statement

Where GitHub-based evidence is used, academic credit is awarded for verifiable documented work aligned to course outcomes.

Repository visibility may be public or private unless otherwise stated, but authorized academic reviewers must be able to inspect required evidence.

Open Discussion Pedagogy Note

This course may use structured critique of language technologies, representation limits, and public claims about conversational AI.

Mirror → Water → Fire Alignment

Mirror: represent language data, tokens, and models accurately  
Water: interpret patterns, outputs, and contextual meaning  
Fire: build and document NLP workflows and model experiments
