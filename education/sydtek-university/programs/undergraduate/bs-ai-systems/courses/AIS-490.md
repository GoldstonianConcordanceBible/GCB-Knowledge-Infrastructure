AIS-490 Course Shell
AI Research Proposal

Credits: 3  
Level: Undergraduate

Catalog Description

Prepares students to define, scope, and propose a substantial undergraduate AI project through formal problem framing, literature review, methodology planning, repository design, and assessment criteria.

Students develop the proposal foundation for the capstone sequence and prepare a faculty-reviewable project plan.

Prerequisites

AIS-240  
Senior standing

Course Learning Outcomes

Students will be able to:

1 define a substantial AI project question or system problem
2 conduct a focused literature and precedent review
3 design a methodology appropriate for undergraduate AI research or system development
4 create a repository and documentation plan for capstone evidence
5 produce a formal proposal suitable for faculty review and approval

GitHub Evidence Statement

Where GitHub-based evidence is used, academic credit is awarded for verifiable documented work aligned to course outcomes.

Repository visibility may be public or private unless otherwise stated, but authorized academic reviewers must be able to inspect required evidence.

Open Discussion Pedagogy Note

This course may include inspectable proposal workshops, peer critique, and evidence-linked revision before capstone approval.

Mirror → Water → Fire Alignment

Mirror: represent the problem, sources, and project scope accurately  
Water: interpret feasibility, prior work, and methodological choices responsibly  
Fire: refine and submit a documented proposal for capstone execution
