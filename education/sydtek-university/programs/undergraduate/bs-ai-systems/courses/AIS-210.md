AIS-210 Course Shell
Data Engineering for AI

Credits: 3  
Level: Undergraduate

Catalog Description

Introduces data engineering principles that support artificial intelligence systems including data collection, cleaning, transformation, storage, versioning, and pipeline design.

Students learn how reliable data workflows support reproducible model development and scalable AI systems.

Prerequisites

AIS-110  
AIS-120

Course Learning Outcomes

Students will be able to:

1 explain the role of data engineering in AI system development
2 construct basic workflows for collecting, cleaning, and transforming data
3 evaluate data quality, consistency, and readiness for model use
4 document data lineage and reproducible pipeline steps
5 preserve data workflow evidence through inspectable repository artifacts

GitHub Evidence Statement

Where GitHub-based evidence is used, academic credit is awarded for verifiable documented work aligned to course outcomes.

Repository visibility may be public or private unless otherwise stated, but authorized academic reviewers must be able to inspect required evidence.

Open Discussion Pedagogy Note

This course may include inspectable critique of data quality failures, bias sources, and workflow design decisions.

Mirror → Water → Fire Alignment

Mirror: represent data sources, structures, and transformations accurately  
Water: interpret data quality, variation, and pipeline implications  
Fire: build and document reproducible data workflows for AI systems
