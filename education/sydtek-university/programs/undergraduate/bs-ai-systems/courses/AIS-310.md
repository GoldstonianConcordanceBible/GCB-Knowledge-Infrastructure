AIS-310 Course Shell
AI Agents and Autonomous Systems

Credits: 3  
Level: Undergraduate

Catalog Description

Studies autonomous AI agents, agent architectures, environment interaction, reinforcement learning foundations, planning logic, and multi-agent coordination.

Students examine how intelligent agents perceive, decide, act, and collaborate across dynamic computational environments.

Prerequisites

AIS-220

Course Learning Outcomes

Students will be able to:

1 explain foundational concepts in autonomous and agent-based systems
2 distinguish reactive, goal-oriented, and learning-based agent models
3 analyze basic reinforcement learning and agent-environment interaction
4 evaluate multi-agent coordination, communication, and task structure
5 document agent behavior, system design, and experiment results through inspectable artifacts

GitHub Evidence Statement

Where GitHub-based evidence is used, academic credit is awarded for verifiable documented work aligned to course outcomes.

Open Discussion Pedagogy Note

This course may include structured critique of autonomous system behavior, agent governance, and claims about AI autonomy.

Mirror → Water → Fire Alignment

Mirror: represent agent structures, goals, and environment relationships accurately  
Water: interpret agent behavior, adaptation, and coordination dynamics  
Fire: build and document autonomous agent workflows and system experiments
