# Student Success Support
## BS in Web3 Systems and Decentralized Infrastructure
## SydTek University

This document provides a day-zero support framework for student success.

---

## Purpose

Student support should help learners persist, improve, and complete the program successfully.

---

## Support Areas

Students may need support in areas such as:

- writing
- programming
- mathematics
- research
- documentation
- GitHub use
- time management
- public speaking

---

## Support Options

Possible supports include:

- advising
- tutoring
- writing feedback
- technical bootstrapping
- peer study structures
- milestone check-ins
- capstone mentoring

---

## GitHub Support Note

Because many courses may use GitHub-based evidence, students may require onboarding in:

- repository setup
- commit practices
- documentation habits
- milestone organization
- evidence preservation
