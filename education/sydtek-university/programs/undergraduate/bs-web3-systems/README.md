# Bachelor of Science in Web3 Systems and Decentralized Infrastructure

Institution: SydTek University

This program explores the design, architecture, and governance of decentralized systems.

The degree draws from the book:

Welcome to Web3: A Multi-Level Introduction to the Decentralized Internet.

---

## Degree Focus

Students learn to understand and design:

• blockchain systems  
• decentralized applications  
• token economies  
• decentralized governance models  

---

## Program Structure

Major Core  
Technical Support Core  
Mathematics and Analytics Core  
Communication and Ethics Core  
Senior Capstone

---

## Program Goal

Graduates of the program will be able to design and analyze decentralized technological systems.
