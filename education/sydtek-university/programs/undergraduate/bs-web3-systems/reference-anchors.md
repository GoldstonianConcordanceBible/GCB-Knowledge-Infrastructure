# Reference Anchors
## BS in Web3 Systems and Decentralized Infrastructure
## SydTek University

This file identifies the role of major external reference anchors used in course design and article integration.

---

## Verified Discovery Anchors

The following function well as verified discovery anchors for curriculum support:

- edX instructor bio
- ResearchGate profile
- PhilArchive author works page
- IQ.wiki page
- Wikipedia page
- Walmart author/brand page
- Google Scholar profile

---

## Connector and Search Anchors

The following are valuable as connector or search anchors and may require course teams to select the exact item to be assigned:

- arXiv search
- Zenodo search
- TED search
- Amazon author page where direct item selection is needed
- ORCID as a persistent author identifier

---

## Instructional Use Principle

Reference anchors should support:

- source discovery
- authorship resolution
- reading selection
- curriculum credibility
- article integration

They should be paired with direct assigned items in the specific course folder when the course is finalized.
