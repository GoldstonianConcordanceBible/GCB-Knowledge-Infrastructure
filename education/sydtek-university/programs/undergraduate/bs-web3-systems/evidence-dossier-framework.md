# Evidence Dossier Framework
## BS in Web3 Systems and Decentralized Infrastructure

Institution: SydTek University

This document defines the student-level evidence dossier used to preserve learning records.

## Purpose

The evidence dossier preserves a documented trail of student learning across assignments, revisions, critiques, and final artifacts.

## Typical Evidence Dossier Contents

A dossier may include:

- student identifier
- course identifier
- repository link
- selected commit history
- artifact snapshots
- reflection memos
- peer or instructor feedback
- rubric scores
- revision notes
- final submission record

## GitHub Requirement

In GitHub-based courses, the evidence dossier should include enough repository evidence to verify:

- authorship
- progression
- milestone completion
- final artifact integrity

## Public and Private Pathways

Evidence dossiers may be built from either public or private repositories as long as authorized academic reviewers can inspect the required evidence.
