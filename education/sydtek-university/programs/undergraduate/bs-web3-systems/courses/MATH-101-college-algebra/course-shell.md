# MATH-101 Course Shell
## College Algebra

Credits: 3  
Level: Undergraduate

## Catalog Description

Develops algebraic reasoning, equations, functions, graphs, and quantitative problem solving required for technical undergraduate study.

## Prerequisites

None

## Course Learning Outcomes

Students will be able to:

1. solve algebraic equations and inequalities
2. interpret functions and graphs
3. apply algebraic reasoning to technical problems
4. organize quantitative work clearly
5. communicate mathematical reasoning in a structured way

## Mirror → Water → Fire Alignment

Mirror:
state the mathematical problem accurately

Water:

Fire:
