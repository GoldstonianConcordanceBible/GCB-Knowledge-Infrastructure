# COMM-101 Course Shell
## Public Communication

Credits: 3  
Level: Undergraduate

## Catalog Description

Develops oral communication, presentation structure, audience awareness, and disciplined public explanation of ideas.

## Prerequisites

None

## Course Learning Outcomes

Students will be able to:

1. organize and deliver structured presentations
2. communicate to varied audiences clearly
3. support spoken claims with evidence
4. revise communication based on feedback
5. preserve communication artifacts for academic review

## GitHub Evidence Statement

In GitHub-based assignments, academic credit depends on verifiable documented work aligned to course outcomes. Public or private repositories may be used when authorized academic reviewers can inspect required evidence.

## Open Discussion Pedagogy Note

This course is naturally compatible with inspectable discourse, critique, and revision cycles.

## Mirror → Water → Fire Alignment

Mirror:
state the message and evidence accurately

Water:
adapt the communication to audience and context

Fire:
refine delivery and produce a persuasive final presentation
