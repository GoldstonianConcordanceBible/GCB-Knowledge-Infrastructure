# INT-480 Course Shell
## Internship / Applied Experience

Credits: 3  
Level: Undergraduate

## Catalog Description

Supervised applied-learning experience in which students contribute to a technical, research, educational, or project environment relevant to program goals.

## Prerequisites

Junior or senior standing  
program approval

## Course Learning Outcomes

Students will be able to:

1. apply academic knowledge in a supervised setting
2. communicate professionally about work and responsibilities
3. document contributions and growth over time
4. reflect critically on strengths, limits, and next steps
5. present evidence of applied experience for academic review

## GitHub Evidence Statement

Where GitHub or version-controlled project evidence is used, academic credit is awarded for verifiable documented work aligned to course outcomes.

Repository visibility may be public or private unless otherwise stated, but authorized academic reviewers must be able to inspect required evidence.

## Open Discussion Pedagogy Note

This course may include structured reflection, mentor feedback, and inspectable documentation of applied learning.

## Mirror → Water → Fire Alignment

Mirror:
state the work context and responsibilities accurately

Water:
contextualize the experience, challenges, and learning

Fire:
synthesize the applied learning and present a refined reflection
