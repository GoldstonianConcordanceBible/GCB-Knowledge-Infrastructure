# WEB3-360 Course Shell
## Web3 Security and Risk

Credits: 3  
Level: Undergraduate

## Catalog Description

Examines security threats, contract vulnerabilities, governance capture, custody risks, operational failures, and trust breakdowns in decentralized systems.

## Prerequisites

WEB3-301  
WEB3-320

## Course Learning Outcomes

Students will be able to:

1. identify major categories of Web3 risk
2. analyze security and governance failure scenarios
3. evaluate basic protection strategies
4. communicate risk clearly in technical and institutional terms
5. document risk analysis with verifiable evidence

## GitHub Evidence Statement

In GitHub-based assignments, academic credit is awarded for verifiable documented work aligned to course outcomes. Public and private repositories are both acceptable when authorized academic reviewers can inspect required evidence.

## Open Discussion Pedagogy Note

This course may use case-based critique and structured response exercises around security failures.

## Mirror → Water → Fire Alignment

Mirror:
identify the vulnerability or failure accurately

Water:
interpret technical and organizational context

Fire:
synthesize mitigation and risk judgment
