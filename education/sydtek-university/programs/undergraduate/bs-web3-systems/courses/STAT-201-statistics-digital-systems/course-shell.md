# STAT-201 Course Shell
## Statistics for Digital Systems

Credits: 3  
Level: Undergraduate

## Catalog Description

Introduces descriptive statistics, probability, distributions, inference, and data interpretation for technical and digital systems.

## Prerequisites

MATH-101

## Course Learning Outcomes

Students will be able to:

1. summarize data using descriptive statistics
2. explain basic probability concepts
3. interpret simple statistical results
4. apply statistical reasoning to digital or technical contexts
5. document analysis in a clear and reviewable format
