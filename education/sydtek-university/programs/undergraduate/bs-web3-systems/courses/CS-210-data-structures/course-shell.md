# CS-210 Course Shell
## Data Structures

Credits: 3  
Level: Undergraduate

## Catalog Description

Introduces essential data structures, storage patterns, basic algorithmic reasoning, and organizational logic for computational systems.

## Prerequisites

CS-101

## Course Learning Outcomes

Students will be able to:

1. identify common data structures
2. explain how data structures support efficient problem solving
3. compare storage and access patterns
4. apply algorithmic reasoning to simple technical tasks
5. document code and analysis clearly

## GitHub Evidence Statement

Where repository-based work is required, academic credit is awarded for verifiable documented work aligned to course outcomes. Repository visibility may be public or private unless otherwise stated, but authorized academic reviewers must be able to inspect required evidence.
