# CS-120 Course Shell
## Web Development Foundations

Credits: 3  
Level: Undergraduate

## Catalog Description

Introduces foundational web development including page structure, styling, basic scripting, interface logic, and front-end organization.

## Prerequisites

CS-101 or program approval

## Course Learning Outcomes

Students will be able to:

1. explain how web pages are structured
2. build simple client-side interfaces
3. apply basic styling and scripting
4. organize front-end files logically
5. document development work through structured artifacts

## GitHub Evidence Statement

In GitHub-based assignments, academic credit depends on verifiable documented work aligned to course outcomes. Public and private repositories are both acceptable when authorized academic reviewers can inspect required evidence.

## Open Discussion Pedagogy Note

This course may include critique and revision of interface design artifacts.

## Mirror → Water → Fire Alignment

Mirror:
represent interface goals accurately

Water:
contextualize usability and front-end structure

Fire:
revise, refine, and deliver a functioning interface
