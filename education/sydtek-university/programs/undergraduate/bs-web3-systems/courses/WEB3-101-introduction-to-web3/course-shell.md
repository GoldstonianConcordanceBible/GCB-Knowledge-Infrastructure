# WEB3-101 Course Shell
## Introduction to Web3 Systems

Credits: 3  
Level: Undergraduate

## Catalog Description

Introduces students to foundational Web3 concepts including decentralization, blockchain, digital ownership, tokens, decentralized applications, and governance models.

## Prerequisites

None

## Course Learning Outcomes

Students will be able to:

1. explain the meaning of Web3 and decentralization
2. distinguish Web3 from earlier internet models
3. identify core components of decentralized systems
4. describe basic governance and ownership concepts
5. document foundational learning through structured academic artifacts
