# WEB3-101 GitHub Requirements
## Introduction to Web3 Systems

## Purpose

This course uses GitHub as an evidence environment for selected assignments and final documentation.

---

## Core Requirement

Academic credit is awarded for verifiable documented work aligned to course outcomes.

Repositories may be public or private unless otherwise stated, but authorized academic reviewers must be able to inspect required evidence.

---

## Minimum Expectations

Students should maintain:

- a clear README
- organized assignment files
- meaningful commit history
- a final folder or clearly marked final submissions
- short reflection notes where required

---

## Recommended Basic Structure

- README.md
- assignments/
- reflections/
- final/
- evidence-notes/

---

## Visibility

Public repositories may support portfolio development.

Private repositories remain acceptable when permitted, provided that faculty can review the required evidence.
