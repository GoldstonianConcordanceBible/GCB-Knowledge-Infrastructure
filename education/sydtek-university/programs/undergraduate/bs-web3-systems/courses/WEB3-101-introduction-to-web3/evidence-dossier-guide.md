# WEB3-101 Evidence Dossier Guide
## Introduction to Web3 Systems

## Minimum Dossier Contents

- student identifier
- course code
- repository link
- repository visibility note
- selected commits or milestone references
- concept map or equivalent artifact
- midterm synthesis
- final Web3 systems brief
- rubric scores
- instructor feedback

---

## Purpose

The dossier should make it possible for an authorized reviewer to see:

- what the student completed
- how the work progressed
- how the work aligned with outcomes
- how the work was evaluated
