# WEB3-101 Rubric
## Introduction to Web3 Systems

## Rating Scale

4 = Exceeds Expectations  
3 = Meets Expectations  
2 = Approaches Expectations  
1 = Does Not Meet Expectations

---

## Criteria

### Concept Accuracy
The student explains Web3-related concepts accurately.

### Comparative Understanding
The student distinguishes Web3 from earlier internet models clearly.

### Evidence and Documentation
The student provides organized, reviewable, and course-appropriate documentation.

### Reasoning Quality
The student demonstrates thoughtful interpretation rather than unsupported slogan use.

### GitHub Evidence Quality
Where applicable, the student provides verifiable repository evidence showing authorship and progression.

### Communication Quality
The student communicates clearly and appropriately for the assignment.
