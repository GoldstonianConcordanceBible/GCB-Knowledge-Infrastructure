# WEB3-340 Course Shell
## DAO Governance and Decentralized Organizations

Credits: 3  
Level: Undergraduate

## Catalog Description

Studies decentralized autonomous organizations, governance processes, proposal systems, treasury stewardship, delegation, participation, and organizational design.

## Prerequisites

WEB3-320

## Course Learning Outcomes

Students will be able to:

1. define the DAO model and its governance logic
2. describe proposal and voting structures
3. analyze governance weaknesses and participation issues
4. design a mission-aligned governance framework
5. preserve governance work in evidence-linked academic artifacts

## GitHub Evidence Statement

Where repository-based evidence is required, academic credit is awarded for verifiable documented work aligned to course outcomes. Repository visibility may be public or private unless otherwise stated, but authorized academic reviewers must be able to inspect required evidence.

This course may use public or private discourse around governance design, critique, and revision.

Water:

Fire:
