# ENG-101 Course Shell
## Writing for Technical Systems

Credits: 3  
Level: Undergraduate

## Catalog Description

Develops clear, organized, audience-aware writing for technical, academic, and documentation contexts.

## Prerequisites

None

## Course Learning Outcomes

Students will be able to:

1. write clearly for technical audiences
2. organize information logically
3. revise writing for clarity and structure
4. document sources and evidence appropriately
5. produce durable written artifacts suitable for academic review

## GitHub Evidence Statement

In GitHub-based assignments, academic credit is awarded for verifiable documented work aligned to course outcomes. Public and private repositories are both acceptable when authorized academic reviewers can inspect required evidence.

## Open Discussion Pedagogy Note

This course may use public or inspectable writing-based discourse and revision cycles.

## Mirror → Water → Fire Alignment

Mirror:
represent the source or system accurately

Water:
contextualize and organize the explanation

Fire:
revise, sharpen, and finalize the written communication
