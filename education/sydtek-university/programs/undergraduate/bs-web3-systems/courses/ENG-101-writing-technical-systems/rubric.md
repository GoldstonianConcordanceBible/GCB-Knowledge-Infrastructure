# ENG-101 Rubric
## Writing for Technical Systems

## Rating Scale

4 = Exceeds Expectations  
3 = Meets Expectations  
2 = Approaches Expectations  
1 = Does Not Meet Expectations

---

## Criteria

### Clarity
The student writes clearly and understandably.

### Organization
The student structures ideas logically and coherently.

### Accuracy
The student represents concepts and sources faithfully.

### Revision Quality
The student improves the work meaningfully through revision.

### Evidence and Documentation
The student preserves organized, reviewable written artifacts.

### GitHub Evidence Quality
Where applicable, the student provides verifiable repository evidence showing authorship and progression.
