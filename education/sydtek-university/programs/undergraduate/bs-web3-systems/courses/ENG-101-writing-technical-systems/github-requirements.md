# ENG-101 GitHub Requirements
## Writing for Technical Systems

## Purpose

This course may use GitHub as an evidence environment for preserving drafts, revisions, reflections, and final portfolio materials.

---

## Core Requirement

Academic credit is awarded for verifiable documented work aligned to course outcomes.

Repositories may be public or private unless otherwise stated, but authorized academic reviewers must be able to inspect required evidence.

---

## Minimum Expectations

Students should preserve:

- drafts
- revised drafts
- final versions
- reflection notes
- portfolio files
- README or navigation note

---

## Recommended Structure

- README.md
- drafts/
- revisions/
- reflections/
- final-portfolio/
- evidence-notes/

---

## Review Principle

Credit is awarded for documented writing development and completion, not repository visibility alone.
