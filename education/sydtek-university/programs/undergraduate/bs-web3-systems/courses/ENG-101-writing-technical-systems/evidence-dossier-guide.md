# ENG-101 Evidence Dossier Guide
## Writing for Technical Systems

## Minimum Dossier Contents

- student identifier
- course code
- repository link
- visibility note
- selected draft milestones
- selected revised pieces
- revision memo
- final writing portfolio
- rubric scores
- instructor feedback

---

## Purpose

The dossier should allow an authorized reviewer to see:

- what the student wrote
- how the writing changed
- how revision improved the work
- how the work aligned with course outcomes
