# MATH-210 Course Shell
## Discrete Mathematics

Credits: 3  
Level: Undergraduate

## Catalog Description

Introduces logic, sets, relations, combinatorics, graph structures, and discrete reasoning relevant to technical systems and computing.

## Prerequisites

MATH-101

## Course Learning Outcomes

Students will be able to:

1. explain basic discrete mathematical structures
2. apply logical reasoning to technical problems
3. analyze relations, sets, and graph-based models
4. interpret combinatorial problems
5. document mathematical reasoning and structure clearly
