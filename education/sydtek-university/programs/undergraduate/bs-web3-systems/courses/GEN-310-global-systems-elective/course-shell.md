# GEN-310 Course Shell
## Global Systems Elective

Credits: 3  
Level: Undergraduate

## Catalog Description

Examines global systems, interdependence, transnational structures, and large-scale coordination across technological, economic, social, or political domains.

## Prerequisites

None

## Course Learning Outcomes

Students will be able to:

1. explain key features of global systems
2. analyze interdependence and systemic complexity
3. compare multiple perspectives on global coordination
4. communicate structured analysis of global issues
5. preserve coursework in a documented and reviewable format

## GitHub Evidence Statement

Where GitHub-based evidence is required, academic credit is awarded for verifiable documented work aligned to course outcomes. Repository visibility may be public or private unless otherwise stated, but authorized academic reviewers must be able to inspect required evidence.

## Open Discussion Pedagogy Note

This course may use inspectable discussion and evidence-linked comparative analysis.
