# WEB3-420 Course Shell
## Applied Web3 Studio

Credits: 3  
Level: Undergraduate

## Catalog Description

Studio-based course in which students design, prototype, document, and iteratively improve a Web3-related system, artifact, or workflow.

## Prerequisites

WEB3-301  
WEB3-320  
program approval recommended

## Course Learning Outcomes

Students will be able to:

1. scope and structure a studio-scale Web3 project
2. produce iterative design artifacts and milestone outputs
3. integrate technical, economic, and documentation decisions
4. revise work in response to critique
5. maintain a professional evidence trail for project review

## GitHub Evidence Statement

This course normally uses GitHub as a primary evidence environment.

Academic credit is awarded for verifiable documented work aligned to course outcomes. Repository visibility may be public or private unless otherwise stated, but authorized academic reviewers must be able to inspect required evidence.

## Open Discussion Pedagogy Note

This studio may include public or private critique cycles, portfolio-facing outputs, and evidence-linked revision.

## Mirror → Water → Fire Alignment

Mirror:
represent the project problem and design target accurately

Water:
contextualize system decisions, constraints, and alternatives

Fire:
iterate, refine, and deliver a stronger final artifact
