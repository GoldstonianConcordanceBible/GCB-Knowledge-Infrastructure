# WEB3-320 Course Shell
## Tokenomics and Digital Economies

Credits: 3  
Level: Undergraduate

## Catalog Description

Introduces token-based incentive systems, supply models, governance tokens, treasury design, and sustainability issues in decentralized digital economies.

## Prerequisites

WEB3-301

## Course Learning Outcomes

Students will be able to:

1. define tokenomics and explain its function
2. compare token types and incentive structures
3. analyze token supply and allocation decisions
4. evaluate sustainability and governance risks
5. document economic reasoning and revision through evidence-linked work

## Open Discussion Pedagogy Note

This course may use evidence-linked public or private critique of token models.

## Mirror → Water → Fire Alignment

Mirror:
state the token model accurately

Water:
contextualize incentives, allocations, and ecosystem behavior

Fire:
evaluate weaknesses, improve the model, and synthesize conclusions
