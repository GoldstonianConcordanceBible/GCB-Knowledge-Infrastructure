# ETH-220 Course Shell
## Ethics of Decentralized Technology

Credits: 3  
Level: Undergraduate

## Catalog Description

Examines ethical questions in decentralized systems, AI, digital ownership, governance, access, responsibility, and technological power.

## Prerequisites

None

## Course Learning Outcomes

Students will be able to:

1. identify ethical issues in decentralized technology
2. analyze competing values and interests
3. apply ethical reasoning to technical systems
4. communicate ethical judgments responsibly
5. preserve evidence-linked ethical analysis for review

## GitHub Evidence Statement

Where repository-based work is required, academic credit is awarded for verifiable documented work aligned to course outcomes. Repository visibility may be public or private unless otherwise stated, but authorized academic reviewers must be able to inspect required evidence.

## Open Discussion Pedagogy Note

This course may use structured public or private discourse on contested ethical issues.

## Mirror → Water → Fire Alignment

Mirror:
represent the ethical issue accurately

Water:
contextualize stakeholders, values, and consequences

Fire:
refine the analysis and defend a reasoned judgment
