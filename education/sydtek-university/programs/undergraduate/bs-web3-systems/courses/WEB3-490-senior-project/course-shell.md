# WEB3-490 Course Shell
## Senior Project in Web3 Systems

Credits: 3  
Level: Undergraduate

## Catalog Description

Capstone course in which students integrate architecture, execution, economics, governance, risk awareness, and professional communication into a final Web3 project.

## Prerequisites

Completion of major core coursework or program approval

## Course Learning Outcomes

Students will be able to:

1. define and scope a substantial Web3 project
2. integrate multiple dimensions of decentralized systems into one coherent design
3. document development through milestone-based evidence
4. present and defend the project professionally
5. produce a capstone-ready artifact portfolio and evidence dossier

## GitHub Evidence Statement

Capstone students must complete project development in a GitHub repository or approved equivalent version-controlled environment.

Academic credit is awarded only when authorized reviewers can verify authorship, progression, milestone completion, and final artifact integrity.

Repository visibility may be public or private unless the capstone track explicitly requires public engagement.

## Open Discussion Pedagogy Note

This course may include public or private critique cycles, provided that equivalent rigor and evidence standards are preserved.

## Mirror → Water → Fire Alignment

Mirror:
define the project problem, source base, and design objective accurately

Water:
contextualize system choices, constraints, and comparisons

Fire:
refine, integrate, defend, and conclude the final project
