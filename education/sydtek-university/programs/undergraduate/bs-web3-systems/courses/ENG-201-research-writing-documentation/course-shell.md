# ENG-201 Course Shell
## Research Writing and Documentation

Credits: 3  
Level: Undergraduate

## Catalog Description

Builds research writing skill through source integration, citation, evidence use, argument structure, and formal documentation.

## Prerequisites

ENG-101  
RES-201

## Course Learning Outcomes

Students will be able to:

1. integrate credible sources into structured writing
2. document evidence appropriately
3. write evidence-based arguments
4. revise research writing for coherence and rigor
5. preserve documented academic writing for review

## GitHub Evidence Statement

Where GitHub-based evidence is used, academic credit is awarded for verifiable documented work aligned to course outcomes. Repository visibility may be public or private unless otherwise stated, but authorized academic reviewers must be able to inspect required evidence.

## Open Discussion Pedagogy Note

This course may include structured evidence-linked discourse and revision.

## Mirror → Water → Fire Alignment

Mirror:
represent sources and claims faithfully

Water:
contextualize evidence and build the argument

Fire:
revise, synthesize, and finalize the documented position
