# CS-240 Course Shell
## Databases and Data Modeling

Credits: 3  
Level: Undergraduate

## Catalog Description

Introduces data organization, schemas, relational structure, normalization, queries, and foundational data-modeling practices.

## Prerequisites

CS-101

## Course Learning Outcomes

Students will be able to:

1. explain basic database concepts
2. model simple relational data structures
3. describe schema and normalization logic
4. write and interpret basic queries
5. document data design decisions clearly

This course may include inspectable schema critique and data-model revision exercises.
