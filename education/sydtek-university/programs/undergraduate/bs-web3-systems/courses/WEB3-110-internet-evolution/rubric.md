# WEB3-110 Rubric
## Internet Evolution: Web1, Web2, and Web3

## Rating Scale

4 = Exceeds Expectations  
3 = Meets Expectations  
2 = Approaches Expectations  
1 = Does Not Meet Expectations

---

## Criteria

### Historical Accuracy
The student represents each internet phase accurately.

### Comparative Analysis
The student compares phases clearly and insightfully.

### Social and Institutional Reasoning
The student explains implications for control, ownership, and participation.

### Evidence and Documentation
The student provides organized, reviewable work aligned to course expectations.

### GitHub Evidence Quality
Where applicable, the student provides verifiable repository evidence showing authorship and progression.

### Communication Quality
The student writes clearly and coherently.
