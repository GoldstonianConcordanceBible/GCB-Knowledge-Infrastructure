# WEB3-110 Evidence Dossier Guide
## Internet Evolution: Web1, Web2, and Web3

## Minimum Dossier Contents

- student identifier
- course code
- repository link
- visibility note
- selected commits or milestone references
- comparison assignments
- midterm comparative analysis
- final analysis paper
- rubric scores
- instructor feedback

---

## Purpose

The dossier should allow an authorized reviewer to understand:

- what the student completed
- how the student’s analysis progressed
- how the work aligned with course outcomes
- how the work was evaluated
