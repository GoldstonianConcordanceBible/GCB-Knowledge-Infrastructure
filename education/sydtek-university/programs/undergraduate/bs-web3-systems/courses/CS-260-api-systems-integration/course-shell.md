# CS-260 Course Shell
## API and Systems Integration

Credits: 3  
Level: Undergraduate

## Catalog Description

Explores APIs, request-response logic, integration workflows, service communication, and interoperability between systems.

## Prerequisites

CS-120  
CS-240

## Course Learning Outcomes

Students will be able to:

1. explain how APIs support system communication
2. describe request-response interactions
3. analyze integration workflows
4. identify basic interoperability challenges
5. document system integration evidence clearly

## GitHub Evidence Statement

Where GitHub-based evidence is required, academic credit is awarded for verifiable documented work aligned to course outcomes. Repository visibility may be public or private unless otherwise stated, but authorized academic reviewers must be able to inspect required evidence.

## Open Discussion Pedagogy Note

This course may include structured analysis of integration scenarios and design tradeoffs.

## Mirror → Water → Fire Alignment

Mirror:
represent the systems and endpoints accurately

Water:
contextualize workflow and integration dependencies

Fire:
refine the integration model and justify the design
