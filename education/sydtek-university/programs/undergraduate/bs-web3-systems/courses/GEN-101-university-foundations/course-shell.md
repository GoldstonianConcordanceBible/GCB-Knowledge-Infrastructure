# GEN-101 Course Shell
## University Foundations

Credits: 3  
Level: Undergraduate

## Catalog Description

Introduces students to university-level learning, academic expectations, documentation habits, time management, and responsible participation in scholarly environments.

## Prerequisites

None

## Course Learning Outcomes

Students will be able to:

1. explain core expectations of undergraduate study
2. organize academic work responsibly
3. use institutional learning systems appropriately
4. reflect on habits required for success
5. preserve academic work in a reviewable format

## GitHub Evidence Statement

Where GitHub-based evidence is used, academic credit is awarded for verifiable documented work aligned to course outcomes. Repository visibility may be public or private unless otherwise stated, but authorized academic reviewers must be able to inspect required evidence.

## Open Discussion Pedagogy Note

This course may introduce students to evidence-linked learning, documentation practice, and critique-ready academic habits.

## Mirror → Water → Fire Alignment

Mirror:
identify the task, expectation, or academic standard accurately

Water:
contextualize how the standard applies to student practice

Fire:
refine habits and produce a documented success plan
