# WEB3-201 Course Shell
## Blockchain Architecture and Distributed Systems

Credits: 3  
Level: Undergraduate

## Catalog Description

Introduces blockchain structure, distributed systems, consensus mechanisms, node roles, transaction flow, and architectural tradeoffs in decentralized networks.

## Prerequisites

WEB3-101  
WEB3-110

## Course Learning Outcomes

Students will be able to:

1. describe the structure of distributed ledgers
2. explain node and validator roles
3. compare major consensus mechanisms
4. analyze scalability, security, and decentralization tradeoffs
5. document systems analysis through evidence-linked artifacts
