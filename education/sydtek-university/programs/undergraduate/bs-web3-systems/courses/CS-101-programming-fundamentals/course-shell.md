# CS-101 Course Shell
## Programming Fundamentals

Credits: 3  
Level: Undergraduate

## Catalog Description

Introduces foundational programming concepts including variables, logic, conditionals, loops, functions, debugging, and structured problem solving.

## Prerequisites

None

## Course Learning Outcomes

Students will be able to:

1. explain foundational programming concepts
2. write basic structured programs
3. use control flow and functions appropriately
4. debug and revise simple code
5. document programming work in an organized academic format

## GitHub Evidence Statement

Where GitHub-based evidence is required, academic credit is awarded for verifiable documented work aligned to course outcomes.

Repository visibility may be public or private unless otherwise stated, but authorized academic reviewers must be able to inspect required evidence.

## Open Discussion Pedagogy Note

This course may include inspectable technical reflection and code explanation activities.

## Mirror → Water → Fire Alignment

Mirror:
represent the problem and code intent accurately

Water:
interpret program flow and revise with context

Fire:
debug, refine, and produce a working solution
