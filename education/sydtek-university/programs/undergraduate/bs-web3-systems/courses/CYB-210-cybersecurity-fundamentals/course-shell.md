# CYB-210 Course Shell
## Cybersecurity Fundamentals

Credits: 3  
Level: Undergraduate

## Catalog Description

Introduces access control, threat awareness, security principles, foundational vulnerabilities, and risk-conscious technical practice.

## Prerequisites

None

## Course Learning Outcomes

Students will be able to:

1. explain foundational cybersecurity concepts
2. identify common categories of technical risk
3. describe basic protection strategies
4. analyze simple threat scenarios
5. document security reasoning and findings clearly

## GitHub Evidence Statement

In GitHub-based assignments, academic credit is awarded for verifiable documented work aligned to course outcomes. Public and private repositories are both acceptable when authorized academic reviewers can inspect required evidence.

## Open Discussion Pedagogy Note

This course may use case-based critique and structured response around security failures.

## Mirror → Water → Fire Alignment

Mirror:
identify the threat or weakness accurately

Water:
interpret the context and risk environment

Fire:
propose mitigation and justify the response
