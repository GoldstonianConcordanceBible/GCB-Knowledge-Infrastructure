# GCB-501 — Advanced Soteriology

Institution: SydTek University  
Program: Master of Theological Systems (MTS)  
Credits: 3  
Level: Graduate  

## Course Description
This course examines salvation as a theological system through scripture, doctrinal history, and comparative interpretation. Students analyze grace, faith, works, justification, sanctification, and assurance across multiple Christian traditions while building structured theological arguments.

## Learning Outcomes
Students will be able to:
- compare major doctrines of salvation
- analyze scriptural foundations for soteriology
- construct a coherent salvation framework
- evaluate tensions between grace, works, and faith
- produce publishable theological analysis

## Major Outputs
- comparative doctrine paper
- salvation framework model
- structured scripture map

## Mirror → Water → Fire Alignment
Mirror: represent scriptural texts accurately  
Water: compare theological positions  
Fire: produce a usable doctrinal framework
