# Grading Framework
## SydTek University

This document provides a day-zero grading framework for undergraduate programs.

---

## Grading Philosophy

Grades should reflect the quality of student learning and demonstrated performance relative to course outcomes.

---

## Typical Grading Components

Courses may include:

- quizzes or exams
- exercises
- labs
- written assignments
- projects
- presentations
- participation

---

## Sample Letter Scale

A = Excellent  
B = Good  
C = Satisfactory  
D = Minimal Passing  
F = Failing

Detailed grading criteria should be provided in each course syllabus.
