# Repository Review Procedure
## SydTek University

This procedure defines how authorized academic reviewers inspect GitHub-based student work.

## Purpose

To ensure that GitHub-based coursework is reviewed consistently, fairly, and in a way that supports academic quality and accreditation-oriented documentation.

## Review Scope

Reviewers may inspect:

- repository structure
- commit history
- dates of progress
- milestone completion
- branch use where applicable
- issue and pull request activity where applicable
- documentation quality
- final artifact integrity

## Visibility

Repositories may be public or private.

Private repositories must provide appropriate access to authorized academic reviewers.

## Core Principle

Academic credit depends on verifiable documented work aligned to course outcomes.

Visibility alone does not determine academic validity.

## Reviewer Notes

Reviewers should record:

- whether evidence was sufficient
- whether authorship appeared credible
- whether progression was observable
- whether required artifacts were present
- whether further clarification was needed
