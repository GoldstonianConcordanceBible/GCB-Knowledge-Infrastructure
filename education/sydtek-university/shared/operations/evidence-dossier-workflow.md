# Evidence Dossier Workflow
## SydTek University

This workflow explains how student evidence dossiers should be assembled and preserved.

---

## Purpose

The evidence dossier preserves student learning in a reviewable form.

It supports:

- grading
- appeals
- capstone review
- portfolio review
- audit preparation
- accreditation-oriented documentation

---

## Workflow Steps

### Step 1 — Identify Required Evidence
Determine which artifacts must be preserved for the course, project, studio, or capstone.

### Step 2 — Capture Repository Evidence
Collect relevant repository information such as:

- link
- visibility type
- milestone commits
- tagged releases where applicable
- screenshots or exports where needed

### Step 3 — Add Course Artifacts
Include:

- written work
- slides
- diagrams
- reflections
- presentations
- rubric sheets
- instructor feedback

### Step 4 — Align to Outcomes
Record which course or program outcomes are represented by the artifacts.

### Step 5 — Preserve the Dossier
Store the dossier in the approved academic or institutional evidence location.

---

## Minimum Dossier Standard

A minimum dossier should allow an authorized reviewer to see:

- who completed the work
- what was submitted
- how the work progressed
- how it was assessed
- how it aligned with outcomes
