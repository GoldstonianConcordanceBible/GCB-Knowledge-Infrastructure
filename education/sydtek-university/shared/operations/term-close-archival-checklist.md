# Term-Close Archival Checklist
## SydTek University

This checklist helps preserve key academic evidence at the close of a term.

---

## Course-Level Closeout

Confirm that the following are preserved where required:

- syllabus
- assignment set
- rubric set
- final grade records
- selected evidence dossiers
- selected audit materials
- improvement notes

---

## GitHub-Based Evidence Closeout

If the course used GitHub-based evidence, confirm preservation of:

- repository links
- milestone snapshots or exports where needed
- final artifact references
- reviewer notes
- access-sensitive records for private repositories where needed

---

## Program-Level Relevance

Important materials should be preserved in a way that supports:

- annual review
- student appeals
- portfolio verification
- capstone verification
- accreditation-oriented documentation

---

## Final Check

A term should be considered archivally complete when required learning evidence and review documentation have been preserved in approved locations.
