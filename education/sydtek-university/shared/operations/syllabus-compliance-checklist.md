# Syllabus Compliance Checklist
## SydTek University

This checklist helps ensure that each course syllabus includes the minimum required academic and operational elements.

---

## Required Syllabus Elements

Confirm that the syllabus includes:

- course code
- course title
- credit hours
- program affiliation where applicable
- course description
- learning outcomes
- major assessments
- grading framework
- attendance and participation expectations
- academic integrity statement
- accommodation statement
- instructor or contact information where applicable

---

## GitHub and Evidence Elements

If the course uses GitHub-based evidence, confirm that the syllabus includes:

- GitHub evidence statement
- public/private repository pathway language
- reviewer access expectation
- evidence documentation expectations
- milestone or submission guidance

---

## Open Discussion Pedagogy Elements

If the course uses discourse-based learning, confirm that the syllabus includes:

- discourse purpose
- critique expectations
- evidence-linking expectation
- moderation or conduct expectations
- equivalent rigorous pathway language where appropriate

---

## Mirror → Water → Fire Elements

If used in the course, confirm that prompts, assignments, or rubrics align with:

- Mirror
- Water
- Fire

---

## Final Compliance Check

Before launch, confirm that the syllabus is:

- complete
- internally consistent
- aligned with course outcomes
- aligned with program expectations
- stored in the approved repository or academic file location
