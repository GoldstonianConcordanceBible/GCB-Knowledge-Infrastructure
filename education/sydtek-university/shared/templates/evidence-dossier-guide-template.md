## Purpose

This guide explains what must be preserved in the evidence dossier for the course.

---

## Minimum Dossier Contents

- student identifier
- course identifier
- repository link
- repository visibility note
- selected commits or milestone references
- final artifact references
- reflection memo
- rubric scores
- instructor feedback
- outcome alignment note

---

## GitHub Evidence Rule

Where GitHub-based evidence is required, academic credit depends on verifiable documented work aligned to course outcomes.

Repository visibility may be public or private unless otherwise stated, but authorized academic reviewers must be able to inspect required evidence.

---

## Preservation Principle

The evidence dossier should make it possible for an authorized reviewer to see:

- what the student did
- how the work progressed
- how the work was evaluated
- how the work aligned with course outcomes
