# Chunk 4 Implementation Notes

## What this chunk does

This file translates the competency framework into an enforceable evaluation tool.

Previous chunks established:
- the institutional purpose of the Teaching Fellowship,
- the credit mapping structure,
- and the competencies students must demonstrate.

This chunk establishes:
- how faculty score those competencies,
- how passing thresholds are determined,
- how evidence must support scores,
- and how remediation is handled.

## Why this matters for accreditation

Without rubrics, a fellowship model can become subjective and difficult to defend.

This file helps SydTek University show that:
- student achievement is measured consistently,
- faculty judgments are documented,
- standards differ appropriately by academic level,
- and credit decisions are based on evidence.

## How to use this file

This file should serve as the core rubric bank for:
- practicum courses,
- fellowships,
- teaching residencies,
- and doctoral field experiences.

Programs may copy these rubrics into course-level documents and add:
- course-specific categories,
- weighting adjustments,
- or specialized performance descriptors.

Any local adaptation should preserve:
- essential category rules,
- evidence requirements,
- faculty review requirements,
- and academic-level differentiation.
