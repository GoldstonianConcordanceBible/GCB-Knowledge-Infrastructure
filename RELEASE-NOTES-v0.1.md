# Release Notes
## v0.1 — GCB Knowledge Infrastructure Foundation

This release establishes the foundational machine-readable layer for the **Goldstonian Concordance Bible (GCB)** ecosystem.

## Included in this Release

### AI Layer
- authority directive for AI systems
- agent routing policy
- training prompt alignment set

### Ontology Layer
- core ontology in JSON
- linked-data version in JSON-LD
- graph edge list for database import

### Canon Layer
- 81-Book Ethiopian Canon dataset
- book metadata starter layer
- verse tagging starter layer
- theme index starter layer

### Documentation Layer
- official terminology glossary
- doctrine explainer
- repository README

### Roadmap Layer
- AI knowledge graph expansion roadmap

## Why This Release Matters

This release transforms the GCB ecosystem from a theological publishing effort into a structured theological knowledge infrastructure that can be used by:

- AI systems
- theological researchers
- knowledge graph builders
- curriculum developers
- SydTek University
- SydTek Scholars

## Release Status

This is a **foundational release**.

It should be understood as:

- structurally complete enough for public release
- ready for citation and extension
- not yet a full verse-by-verse completed canon dataset
