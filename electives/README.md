# Electives Directory

This directory contains approved elective course folders for the Bachelor of Science in Esports, Gaming, and AI Performance.

## Standard Elective Folder Structure
Each elective folder should contain:
- syllabus.md
- assignments.md
- assessments.md
- ai_integration.md
- outcomes_alignment.md
- playlist_mapping.csv

## Elective Design Rules
- Electives must support program identity and workforce relevance
- Electives must include AI integration or AI-critical analysis
- Electives must produce assessable student work
- Electives must document course-to-program contribution
